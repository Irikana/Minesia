# 0.0.15

## 2026.x.x

***

### 新增内容

新增属性面板系统，玩家可以通过第9槽位的属性面板物品查看详细属性信息：

- 属性面板物品使用 `item_lock` 组件锁定在槽位中，无法移动或丢弃
- 使用 DDUI (ActionFormData) 框架显示属性表单，支持 `header`、`label`、`divider` 等增强特性
- 战斗属性：显示暴击率、暴击伤害加成
- 体力属性：显示当前体力、体力上限加成、消耗倍率、恢复倍率
- 生存属性：显示生命值
- Minesia 等级：显示等级、总经验、升级进度条
- 支持中英文双语显示
- 玩家首次加入时自动获得属性面板物品

新增沙漠神殿系列物品，这些物品都可以在沙漠神殿的战利品箱中找到：

- 沙漠神殿碎片：来自沙漠神殿的神秘水晶，可堆叠50个

- 沙漠暴雪：一把铸造于沙漠中的暴雪之剑，伤害4~7，攻击时有50%概率使目标虚弱I持续3秒、减速III持续1秒

- 沙漠神殿珍珠：副手装备时增加100%最大体力值、50%最大生命值，每秒恢复8点体力值

- 沙漠神殿之眼：副手装备时增加30%暴击率、50%最大体力值，每秒恢复4点体力值

以上新物品均以较小概率在沙漠神殿战利品箱中掉落，同时降低了沙漠行者的掉落概率（从100%降至30%）。

新增动态光源系统，手持光源物品时会自动在玩家周围产生光照效果：

- 支持检测主手和副手的光源物品（副手仅支持自定义物品）
- 支持22种光源物品，包括火把、灯笼、荧石、海晶灯、岩浆桶等
- 光源等级根据物品类型自动匹配（如火把14级、灯笼15级）
- 玩家移动时光源自动跟随，收起物品时光源自动消失
- 性能优化：每2 tick更新一次，避免频繁计算

### 优化改进

删除了 lore 脚本里向物品和武器添加 lore 的代码，因为它们已经通过语言文件配置了。

升级附加包到 Minecraft 基岩版 1.26.10（Tiny Takeover），适配 2026 年新的版本编号系统。

升级 @minecraft/server 脚本 API 从 2.5.0 到 2.6.0，支持最新的稳定版 API。

修复了 35 个物品定义文件的 menu_category group 字段，添加了 minecraft: 命名空间前缀以符合新版本要求。

更新了 35 个物品的图标定义格式，从旧版字符串格式改为新版对象格式，确保与新版本兼容。
