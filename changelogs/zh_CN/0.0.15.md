# 0.0.15

## 2026.4.4

***

### 新增内容

新增属性面板系统，玩家可通过第9槽位的属性面板物品查看暴击率、暴击伤害、体力、生命值、Minesia等级等详细属性信息。面板使用 DDUI 框架显示，支持中英文双语，玩家首次加入时自动获得。

新增饰品栏系统，背包第一行（槽位9-17）作为饰品栏，副手物品可在饰品栏触发效果，且与副手效果叠加。

新增沙漠神殿系列物品，包括沙漠神殿碎片、沙漠暴雪、沙漠神殿珍珠、沙漠神殿之眼，均可在沙漠神殿战利品箱中以较小概率找到。同时降低了沙漠行者的掉落概率（从100%降至30%）。

新增动态光源系统，手持光源物品时自动在玩家周围产生光照效果，支持30种光源物品，光源跟随玩家移动。

### 优化改进

为自定义物品创建了创造模式物品栏分组（匕首、镰刀、特殊武器、货币、饰品、材料），方便玩家查找模组物品。

删除了 lore 脚本中向物品添加 lore 的代码，改为通过语言文件配置。

升级附加包到 Minecraft 基岩版 1.26.10（Tiny Takeover），升级 @minecraft/server 脚本 API 从 2.5.0 到 2.6.0。

修复了 35 个物品定义文件的 menu_category group 字段和图标定义格式，确保与新版本兼容。

### 问题修复

修复了套装效果系统中饰品栏物品效果不叠加的问题，暴击率和体力值百分比加成现在正确工作。

修复了生命值百分比加成计算不正确的问题，使用以2为单位的向下取整，装备切换时生命值不再异常。

修复了暴击音效使用错误的问题，将音效从 `random.orb` 改为 `random.crit`。

修复了装备卸下后生命值异常的问题，当前生命值现在正确保持不变。

修复了物品定义与 item_catalog 配置不一致导致的控制台警告，将 fat_apple 和 cooked_beef_super 的 category 从 equipment 改为 items，升级了 wooden_coin 的格式版本。

修正了货币描述：钻石币改为"非常稀有的宝贵货币"，绿宝石币改为"极其稀有的宝贵货币"，银币改为"较为珍贵的普通货币"。
