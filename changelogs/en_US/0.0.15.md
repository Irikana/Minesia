# 0.0.15

## 2026.4.4

***

### New Features

Added attribute panel system. Players can view detailed attribute information including crit rate, crit damage, stamina, health, and Minesia Level through the attribute panel item in slot 9. The panel uses DDUI framework and supports both Chinese and English. Players automatically receive it on first join.

Added accessory slot system. The first row of inventory (slots 9-17) serves as accessory slots. Offhand items can trigger effects in the accessory slot, stacking with offhand effects.

Added Desert Temple series items, including Desert Temple Fragment, Desert Blizzard, Desert Temple Pearl, and Desert Temple Eye. All can be found in desert temple loot chests with small probability. Also reduced Desert Walker drop rate from 100% to 30%.

Added dynamic lighting system. Light source items automatically generate lighting effects around the player when held, supporting 30 light source items. Light follows player movement.

### Improvements

Created creative mode item tab groups for custom items (daggers, scythes, special weapons, currency, accessories, materials) for easier navigation.

Removed lore script code that adds lore to items, now configured through language files.

Upgraded add-on to Minecraft Bedrock 1.26.10 (Tiny Takeover), upgraded @minecraft/server Script API from 2.5.0 to 2.6.0.

Fixed menu_category group field and icon definition format in 35 item definition files for new version compatibility.

### Bug Fixes

Fixed accessory slot item effects not stacking in set effect system. Crit rate and stamina percentage bonuses now work correctly.

Fixed incorrect health percentage bonus calculation, using floor with unit of 2. Health no longer behaves abnormally when switching equipment.

Fixed crit sound effect using wrong sound, changed from `random.orb` to `random.crit`.

Fixed abnormal health after unequipping gear. Current health now correctly remains unchanged.

Fixed console warnings from item definition inconsistencies with item_catalog configuration. Changed fat_apple and cooked_beef_super category from equipment to items, upgraded wooden_coin format version.

Corrected currency descriptions: Diamond Coin to "very rare precious currency", Emerald Coin to "extremely rare precious currency", Silver Coin to "relatively precious common currency".
