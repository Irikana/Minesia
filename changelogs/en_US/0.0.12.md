# 0.0.12
## 2026.3.5
---
### Added Debug System
Added a debug system that replaces console, error, and warn methods with debug methods to send chat messages to the world, making code debugging more convenient during development
### Stamina System Improvements
Eating food now restores stamina based on hunger restoration, by default restoring 10 stamina per 1 hunger point restored
Walking no longer pauses stamina regeneration, instead regenerates at a rate of 5 per second
After waking up, stamina is restored to maximum stamina
When hunger is 0, stamina regeneration is paused; when hunger is at maximum, stamina regenerates at double the rate
### Other Changes
Now when respawning after death, health and stamina are restored to maximum health and maximum stamina after a short delay
Fixed toy sword lore issue
