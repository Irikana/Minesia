# Minesia 0.0.1-beta-4 Changelog

## 2026.2.24

---

## Refactored Bonus Damage System

- Removed weapon components from all addon weapons, merged base damage into minimum bonus damage
- Removed bonus damage bonuses from all vanilla weapons
- Renamed the entire Bonus Damage System to **Random Damage System**

## Improved ActionBar Display

- Minesia Level, Stamina, and Random Damage can now be displayed simultaneously
- Automatic line breaks based on priority

## New Content

- Added **Copper Dagger**
- Added Lore descriptions for currencies

## Stamina System Optimization

- Optimized stamina display: now continuously displays when stamina changes
- Added Lore to all weapons stating stamina cost per use or attack
  - Displayed below damage values in red

## Balance Adjustments

- Increased damage range (both min and max) by 2 for all Scythes
- Scythes now consume 2 additional stamina points

## Lore Improvements

- Added Lore descriptions for all stamina-consuming weapons
- Added stamina cost Lore for vanilla weapons
