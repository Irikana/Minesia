# Minesia 0.0.1-alpha-2 Changelog

## 2026.2.19

---

## Added Dagger Series Weapons

| Weapon | Damage | Additional Damage |
|--------|--------|-------------------|
| Wooden Dagger | 2 | 0~2 |
| Stone Dagger | 3 | 0~2 |
| Iron Dagger | 4 | 0~2 |
| Steel Dagger | 4 | 0~3 |
| Diamond Dagger | 5 | 0~2 |
| Netherite Dagger | 6 | 0~2 |

- Slightly lower durability than swords
- Crafting: One stick and one corresponding material arranged vertically, stick at bottom

## Stamina System

- Added stamina system with set effect integration
- Running, jumping, and attacking (detected via position changes and `world.afterEvents.entityHurt` event) consume stamina
- Graphical display via ActionBar, same principle as Creation Level
- Display priority: Higher than Creation Level, lower than damage display

## Custom Event System

- Added custom event system, including item custom event system
- Integrated with set effects: custom events trigger when specific sets are activated

## Other Updates

- Slightly increased wood coin requirements in trades, reduced wood coin purchasing power for balance
- Added Golden Phantom Membrane

## Bug Fixes

- Fixed initialization event error
