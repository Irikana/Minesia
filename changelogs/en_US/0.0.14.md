# 0.0.14

## 2026.3.18

***

### New Features

Added critical hit system. Some weapons and equipment now have critical hit rate bonuses!

Item descriptions are now displayed directly in the creative inventory, no need to wait until you get the item to see its attributes.

### Improvements

Optimized equipment detection performance, the game should run smoother.

Fixed the issue where Currency Merchant Lyman couldn't spawn naturally. Now you should be able to encounter Equipment Merchants and Currency Merchants in more places.

Critical hit damage display now supports localization, AOE has added combo detection, and critical hit and combo notifications are clearer.
