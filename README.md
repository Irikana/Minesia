# Minesia

A Minecraft Bedrock Edition addon that creates a new world experience.

## 基本信息

- **作者**: Hua1286
- **版本**: 0.0.13
- **最低引擎版本**: 1.21.130
- **许可证**: CC BY-NC-SA 4.0 (非商业性使用-相同方式共享)

## 核心系统

| 系统      | 描述              |
| ------- | --------------- |
| 套装效果系统  | 装备套装时触发特殊效果     |
| 等级系统    | Minesia 等级与奖励机制 |
| 随机伤害系统  | 武器伤害浮动机制        |
| Lore系统  | 物品描述动态显示        |
| 伤害显示系统  | 显示造成的伤害数值       |
| 体力值系统   | 武器攻击消耗体力        |
| 自定义物品事件 | 特殊物品触发独特效果      |

## 自定义内容

### 武器

- **匕首系列**: 木制、石制、铁制、铜制、金制、钻石、下界合金、钢制
- **镰刀系列**: 木制、石制、铁制、铜制、金制、钻石、下界合金、钢制、沙漠
- **特殊武器**: 黑色匕首、末影珍珠剑、白色金剑、开拓者、自私等

### 装备

- **钢制盔甲**: 钢头盔、钢胸甲、钢护腿、钢靴子

### 特殊物品

- 生命石、雕像图腾、肥苹果、蜘蛛腿、责任之冰、火焰精灵等

### 货币系统

木币 → 石币 → 银币 → 金币 → 绿宝石币 → 钻石币 → 红宝石币

### 自定义实体

- Ryaman - 交易村民
- Victor - 交易村民
