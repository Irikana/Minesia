# 问题分析

## 当前问题
1. **absorption效果的副作用**：会带来金色生命值条，并且会不断补充临时生命值，导致玩家生命值根本不减
2. **之前的问题**：使用health_boost效果时，生命值会锐减
3. **效果持续时间**：过长会导致装备不满足效果很久也不会消失
4. **生命值上限**：在player.json里已经声明过了

## 解决方案

### 1. 重新设计执行流程，使用health_boost效果
- **修改执行顺序**：
  1. 先计算所有新效果
  2. 应用新效果
  3. 清理旧状态
  4. 确保玩家不会处于没有效果的状态

### 2. 实现步骤
1. **修改 `main.js` 文件**：
   - 重新设计执行流程
   - 确保每次都正确计算和应用效果
   - 避免玩家处于没有效果的状态

2. **修改 `actions.js` 文件**：
   - 确保效果的正确应用
   - 避免效果持续时间过长的问题

3. **验证修复**：
   - 测试副手持盾牌主手持水晶碎片的效果
   - 测试主手切换物品时的效果
   - 确保生命值不会锐减
   - 确保装备不满足效果时效果会及时消失

### 3. 预期效果
- **装备紫水晶碎片**：生命值上限从20点增加到22点
- **装备盾牌**：生命值上限从20点增加到22点
- **同时装备两者**：生命值上限从20点增加到24点
- **主手切换物品**：生命值上限平滑变化，不会锐减
- **装备不满足效果**：效果会及时消失
